API Documentation

Educators API End Points

  • GET all Educators - /api/educators
  • GET one Educator Details - /api/educators/:educatorId
  • POST a new Educator - /api/educators. Post Object Structure

    {name: educator-name, email: educator-email, position: educator-position}

  • Update an Educator - /api/educators/:educatorId
  • Delete an Educator - /api/educators/:educatorId

Students API End Points

  • GET all Students - /api/students
  • GET one Student Details - /api/students/:studentId
  • POST a new Student - /api/students. Post Object Structure

    {name: student-name, email: student-email, course: student-course, educator_id: existing-educator-id}

  • Update an Student - /api/students/:studentId
  • Delete an Student - /api/students/:studentId
